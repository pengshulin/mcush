SDIO card example

Example description
This example initializes/enumerates GainSpan SDIO card and will enable users
to interact with the card using the UART. Keystrokes captured from UART is
submitted to the CARD via SDIO and the response from the CARD is printed on
the UART terminal.

Special connection requirements
On the LPC4337 Xpresso shield board JP3 must be soldered to shunt.
